## Overview of ADH Site Audit

ADH Site Audit provides a website audit process built as a static HTML repository. It is intended to work smoothly with GitHub Pages, so publishing and sharing stay simple without needing an extra application layer.

This setup is a practical match for construction-oriented site review, where a compact browser-based layout is often the easiest way to inspect content. Since everything lives in static files, the project remains easy to maintain and distribute with little overhead.
